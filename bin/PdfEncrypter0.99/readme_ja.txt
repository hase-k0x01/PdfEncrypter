# このソフトウェアの説明
このソフトウェアは、PDFファイルを暗号化することができます。

暗号のためのパスワードを設定したり、印刷オプションやアクセシビリティオプションを設定できます。

設定できるPDFオプションは、次の通りです。

 - User Password(ユーザパスワード)
 - Owner Password(所有者パスワード)
 - Permit Accessibility Extract Content(アクセシビリティのための内容抽出を許可)
 - Permit Annotations(アノテーションを許可)
 - Permit Assemble Document(PDFのアセンブルを許可)
 - Permit Extract Content(内容の抽出を許可)
 - Permit Modify Document(内容の変更を許可)
 - Permit Forms Fill(フォーム入力を許可)
 - Permit Full Quality Print(最高画質印刷を許可)
 - Permit Print(印刷を許可)

# 使い方
1. このソフトウェアを起動した後、"Source PDF File Name"入力欄の横にある ... のボタンをクリックしてPDFファイルを選択してください。エクスプローラからPDFファイルをドロップすることもできます。
2. "Destination PDF File Name"入力欄に、暗号化後のファイル名を入力します。
3. "User Password", "Owner Password"の欄にパスワードを入力します。"User Password"を設定すると、PDFファイルをオープンするときにパスワードが要求されます。"Owner Password"を設定すると、PDFファイルを変更しようとするときにパスワードが要求されます。"User Password"を空欄にしておくと、誰でもPDFファイルを開くことができるようになります。
4. "Encrypt Options"のチェックボックスをチェックまたはアンチェックして、有効にするオプションを選択します。
5. "Encryption Strength"から、暗号強度を選択します。
6. "Encrypt"ボタンを押して、PDFを暗号化します。
